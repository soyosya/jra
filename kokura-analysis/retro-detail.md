# 小倉 レース振り返り 完全記録（買目選定の参照資産）

**目的**: 過去の振り返り(各馬の前走・展開/ペース・本命の勝因敗因・波乱要因・決着型)を**今走の分析と買目選定に活かす参照資産**。要約では取りこぼす要素を残すため、**各レースを1レースずつ手抜きなく完全版で記録**する([[keiba-daily-retro-granularity]])。仮説の抽出結果は race-reads.md、完全記録は本ファイル。

**各レースの記録項目(必須)**: ①ヘッダ(R｜種別距離 条件（頭数）｜ベイズ%｜決着型) ②全着順(全頭: 馬番/馬名/四角=脚質/コ順/指数/人気/オッズ) ③全払戻(単複馬連ワイド三連複) ④予想突合(jra軸/コ1位/1番人気→着) ⑤ペース(前走ハナ/先行頭数) ⑥1-3着の前走2-3走 ⑦なぜ(展開/脚質/距離/本命勝因敗因/波乱要因)。
**生データ源**: retro-day.ps1 -Date -Venue → retro-<date>.txt。本ファイル=下段「完全データ(全8日)」＋最下段「なぜ(読み)(全8日)」。

**今走での使い方**: 今走レースの (場×距離×クラス×想定ペース) に該当する過去回を本ファイルから引き、①その条件の決着型(前残り/差し) ②本命/コ1位の信頼度 ③危険パターン(同型過多の逃げ本命・障害の平地本命・芝短フルの紛れ 等) を買目に反映。
