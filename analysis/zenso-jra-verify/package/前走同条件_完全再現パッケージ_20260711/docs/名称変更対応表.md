# 名称変更対応表

## 変更の目的

公開名称を「中団有力馬の確度選抜ロジック」から **「前走同条件」** へ統一した。名称変更に伴い、パッケージ名、正式仕様書名、実行モジュール名、設定ファイル名、監査出力名、CSV/JSONの公開列名、検証資源名を同時に変更した。

## ロジック上の扱い

名称変更のみであり、次の計算は変更していない。

- 対象20セル
- 前走同条件率60%以上
- 有効頭数 `nm >= 5`
- A3-RELによるメンバー内中位1/3
- S1持続条件
- 馬番相対 `< 0.80`
- 同条件組の前走時計1位
- 簡易ADV、簡易HOLD
- S、A-K、A-Uの区分
- 時計同率時は馬番の小さい馬を採用
- 時計1位がM_base外の場合は繰り上げない
- コンピ指数上位4頭と参考買目4点
