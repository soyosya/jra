# 参考買目生成ルール 固定補足仕様 v2.0

本補足は、正式仕様書で「参考表示テンプレート」とされた部分を、第三者との一致検証に使えるよう決定的な処理として固定する。

## 重要な位置付け

- 確立済みなのは前走同条件のS/A確度選抜である。
- 本補足の券種は、オッズ・期待値を使わない **候補表示専用** である。
- 出力された券を「購入推奨」「+EV」と解釈してはならない。
- 前走同条件S/Aが既存主軸と異なっても、自動差替え、自動追加、二頭軸化は行わない。

## コンピ指数スナップショットの正本

`コンピ指数.csv`には同一開催日・場・R・馬番の取得スナップショットが複数入ることがある。各馬について `取得日時` が最大の1行を採用する。同時刻が重複した場合は `Id` が大きい行を採用する。異なるスナップショットを重複馬として順位表へ入れてはならない。

## 基礎印

最新スナップショットへ縮約したコンピ指数をレースごとに次のキーで昇順ソートする。

1. 指数順位（正の値。欠損は最後）
2. 指数の降順
3. 馬番の昇順

上位4頭を順に ◎、○、▲、△ とする。

## 参考買目

- ワイド ◎-○
- ワイド ◎-▲
- 三連複 ◎-○-▲
- 三連複 ◎-○-△

馬番は役割順（◎、○、▲、△）で出力する。三連複でも数値昇順へ並べ替えない。4頭に満たない場合は、必要な役割がそろう券だけ出力する。

## 前走同条件との統合

- Sと◎が同一: 買目は変更せず「確度おすすめ候補」。
- Aと◎が同一: 買目は変更せず「前走同条件A一致候補」。
- Sと◎が別: 「前走同条件S別軸・要確認」。買目は変更しない。
- Aと◎が別: 「前走同条件A別馬・参考」。買目は変更しない。
- 前走同条件選出なし: 変更なし。

## 購入状態

全件 `候補表示のみ（オッズ・EV未使用）` とする。金額は出力しない。
