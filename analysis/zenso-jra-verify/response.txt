2. 指摘1: 簡易ADVの非対応平均
本指摘は正当です。現在のsamecond_logic.pyは、過去3走の3角相対位置平均と4角相対位置平均を、それぞれ別の有効走集合から計算した後に差を取っています。3角のみ存在する走と4角のみ存在する走が混在すると、同一レース内の位置変化を表さない数値になります。これは「3角から4角へ前進したか」を測るADVの意味に反します。
2.1 v2.1で採用する正確な定義
対象馬の直近3走を i=1,2,3 とし、同一過去走について3角相対位置 q3_i と4角相対位置 q4_i の両方が存在する走だけを有効ペア集合 I とします。
I = { i | q3_i と q4_i が同一過去走でともに存在する }d_i = q3_i - q4_iadv_pair_count = |I|adv_pair_mean = mean(d_i for i in I)adv_average_pass = (adv_pair_count >= 1) AND (adv_pair_mean > 0)I が空集合の場合は False ではなく Unknown とする

相対位置は値が小さいほど前方であるため、q3_i - q4_i > 0 は3角から4角で前進したことを意味します。欠損値の補完、別走からの組み合わせ、3角平均と4角平均の別計算は行いません。
2.2 変更しない条件
前走単独のADVは、前走内でp3とp4がともに存在する場合に限り adv_last_pass=(p3-p4)>0 とする。
HOLDは、前走4角相対位置より最終着順相対位置が前であること、すなわち final_rel < p4 を現行どおり使用する。
S判定は adv_average_pass AND adv_last_pass AND hold_pass の3条件をすべて必要とする。
有効ペアが0走の場合はA-Kへ落とさず、A-U（判定不能）として扱う。

3. 協議事項: 時計同率・A3同値のタイブレーク
同値比較器を注入可能な抽象点として設ける提案には賛成します。ただし、時計同率とA3同値を同一の「共通タイブレーク連鎖」で処理することには、仕様上の留保があります。両者は順位付けの目的が異なるからです。

3.1 時計同率に対する回答
馬番昇順が同率馬のうち相対的に低い複勝率の側を選んでいたという全数集計（選択馬47.5%、非選択馬51.6%）は、馬番を能力タイブレークとして使う根拠が弱いことを示しています。この点は受け入れます。時計同率については、正式仕様が提供されれば、次の連鎖をproductionプロファイルへ採用する方向で合意できます。
clock_tie_breaker =  1. 前走 blend 総合スコア上位  2. 前々走 blend 総合スコア上位  3. 馬番昇順

ただし、第三者が同一結果を再生成するには「blend総合スコア」という名称だけでは不足します。正準採用の前提として、次の情報を仕様書・入力スキーマ・エクスポータへ固定してください。
blend総合スコアの計算式、構成項目、係数、丸め前後の精度。
値が大きいほど上位か、小さいほど上位か。
前走・前々走の特定方法と、取消・除外・競走中止の扱い。
欠損時の順序（欠損を最下位とするか、次段へ即移るか）。
同値判定の許容誤差。浮動小数点を完全一致で比較するか、丸め値で比較するか。
対象日発走前に確定している情報だけで計算されること。
CSV列名、型、文字コード、空値表現。
3.2 A3同値に対する反論・留保
A3は位置予測用の指標です。A3が同値の馬をblend能力で並べ替えると、位置が同じと推定された馬の一方を「能力が高い」という理由で中団帯へ入れ、他方を外すことになります。これはA3-RELの位置判定へ能力信号を混入させ、時計1位による後段の能力選抜と二重評価になるおそれがあります。したがって、時計同率用の比較器をA3同値へ自動的に流用することには同意しません。
v2.1では比較器を次の2種類に分離することを提案します。

3.3 A3同値の推奨処理: shared midrank
A3値が同じ馬には、並び順を恣意的に決めず、同順位ブロックが占める順位の平均を全馬へ与えます。現在の rr=(i+0.5)/nm に対応させる場合、同値ブロックの各仮順位に対するrrの平均を使用します。これにより、境界を跨ぐ同値馬が馬番や能力で一部だけ中団へ入ることを防げます。
例: nm=6、A3同値の2頭が仮順位3位・4位を占める場合rr_3 = (2 + 0.5) / 6 = 0.4167rr_4 = (3 + 0.5) / 6 = 0.5833shared_rr = (rr_3 + rr_4) / 2 = 0.5000同値2頭にはともに rr=0.5000 を付与する

同値を全員同順位にするとM_raw頭数が厳密な1/3から増減する場合がありますが、これは「観測上区別できない馬を恣意的に分割しない」という意味で、位置分類としてより整合的です。
