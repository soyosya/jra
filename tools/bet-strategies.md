# JRA 買目ロジック一覧（bet-strategies）

買目（軸・相手・券種）の作り方を名前付きで管理する台帳。新ロジックは行を追加。検証は4開催(2025-06/2026-06/2026-03/2025-10)＝ワイド軸流し基準。すべて回収率<100%（中央は市場効率的＝+EVなし。損失最小化・確度向上が目的）。

## 用語
- **総合スコア** = h2h0.4 + V3(厩舎の話+調教)0.35 + コンピ0.25 + ローテ似0.15（＋前残り場は先行力0.20）のレース内Z合成（jra-card既定）。**要素別の詳細は [eval-criteria.md](eval-criteria.md) の「◎ 総合スコア」節**。
- **ファネル** = ①ネガティブ要素≥2の馬を消し ②残りを総合スコア上位5に絞る ③ポジティブ要素ありに絞る（<2頭なら②のまま）→ 最終プール。
- **コンピ** = コンピ指数の順位（1=最上位）。**ネガ要素** = 消/危/人危/失速/ダ替/連後/▽延/▽遠ロ/遠危/休消/降/調/悪。**ポジ要素** = 連好/連脚/連上/末★/安★/信/矢↗/似(≥35)。

### 各ロジック詳細

**S0 既定ワイド**（jra-weight-loop / jra-export-bets 既定）
- 軸: 総合スコア最上位（危険ラベルで降格あり）。
- 相手: 総合スコア上位3頭。消し・遠危・休消は除外。
- 券種: ワイド軸流し（相手3）。三連複/馬連より回収良([[jra-bettype-roi]])。
- 成績: 2025-06+2026-06の2年プールで回収81%、軸複勝54-60%。
- **定性評価**: 〇シンプルで実装・運用済、軸が総合最上位で確度が高く崩れにくい。✕相手が機械的(総合上位3)でネガ持ち人気馬も混ざり、はずれ点数が多め(1点的中14%)。回収は中位。万人向けの無難な現行運用だが伸びしろは相手選別にあり。

**S1 V3中位×コ5**（最も回収率が高い）
- 軸: ファネル最終プールを**コンピ順位で並べ、中位**の馬。
- 相手: プール内のコンピ≤5の馬（弱い相手＝コンピ6位以下は買わない）。
- 成績: 回収86.8%・4開催全て安定(71-95%)。中位軸×手堅い人気相手のワイドがFLバイアスを突く。
- **定性評価**: 〇4開催で回収最良かつ全開催安定、「中位のやや妙味オッズ軸×複勝圏に来る人気相手」という機構が明快でFLバイアスに根拠あり。✕ファネル＋中位軸とルールが複雑で過学習の懸念が残る(2→4開催で頑健だが要継続監視)、軸自身の複勝率は中程度(47%)で当たりはワイド分散頼み、コンピ中位の選定がプール頭数でブレうる。回収を最優先するなら本命候補。

**S2 V3中位×はずれ最小**（1点が当たりやすい）
- 軸: S1と同じ（コンピ中位）。
- 相手: **コンピ≤3 かつ ポジ要素を1つ以上持つ**馬のみ。
- 成績: 1点的中26.2%(R0全部14.3%の約1.8倍・はずれ率86→74%)・4開催23-29%安定・回収85.1%・点数約1.1/R。※2024-06/2026-01で追加検証中。
- **定性評価**: 〇はずれが最も少なく(1点的中26%)的中感が高い・点数最小(約1.1/R)でコストと精神的負担が軽い・回収もS1とほぼ同等で4開催安定。✕相手を人気上位×好材料に絞るためカバー(買えるレース)が減り的中の総数は減少、大穴は拾えない、回収はS1にわずかに劣る。少点数で当たり頻度・納得感を重視する人に最適。回収最大化には不向き。

**S3 V3中位×回収単発**（回収だけ最高だがブレる）
- 軸: S1と同じ。
- 相手: ネガ要素が一つも無い馬のみ。
- 成績: 回収89.9%だが年別14.7-21.1%とブレ大。安定性はS1/S2に劣る。
- **定性評価**: 〇回収率の瞬間最大は全ロジック中トップ、ネガ無し相手は素直に強い。✕年別の的中・回収のブレが大きく再現性が低い、点数も多め(2.0/R)、単月で良く見え過信を招きやすい。実用は非推奨で参考扱い(ブレ大)。
