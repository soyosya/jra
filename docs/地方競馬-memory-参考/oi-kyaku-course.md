大井のコース・距離別脚質有利度と三→四コーナーまくり分解(2024-01〜2026-06)。スクリプト tools/oi-kyaku-course.ps1。コーナー記録充足: 1200m=C3/C4のみ, 1400m=C4が54%欠損(C3代替), 1600m+=全周完備。

**大井は典型的な前残り(先行有利)。距離が短いほど極端**(逃げのインパクト値=脚質別勝率/全体平均):
- 1200m: 逃げIV3.64(勝率29%)・先行1.84(14.7%)・差し0.67・追込0.16
- 1600m: 逃げIV2.89(25%)・先行1.60・差し0.87・追込0.31
- 1800m: 逃げIV2.42・先行1.70・差し0.95 …延長ほど前優位が緩む

**勝敗を決めるのは序盤位置より「四角位置」(序盤×四角クロス)**:
- 四角で前(top33%)なら距離問わず勝率20%超。
- まくり(序盤中後→四角前)は前残りと互角以上。1600mは後方→四角前でも勝率24.2%/複57.3%。
- ただし**1200mは短すぎて後方→四角前がほぼ不成立**(サンプル僅少)、実質「前→前」か「中→前」のみ。まくりは1600m以上で有効。

**プロファイル改良(検証済み・実装済み)**: 脚質判定を「序盤位置(最初の非ゼロコーナー)」→「四角位置(C4→C3フォールバック)」に変えると、大井1200別定100万のscore4は 頭数380→415(+9%)・勝率22.9%→22.7%(同等)・単回収89.4%→92.7%(+3.3pt)。同精度で該当馬を増やしまくり成功馬を取りこぼさない strictly better。ただし控除率(基準83.5%)は超えず妙味なしは不変([[oi-roi-backtest]]と整合)。
- race-profile-screen.ps1 に -LateCorner スイッチを追加(既定は従来の序盤ベース)。決め脚サブ条件は序盤位置のまま(後方からの差し脚指標として分離)。

**枠順は脚質より遥かに小さい**: 大井1200の枠番IVは最大1.19(1枠)〜0.93(中枠)で弱いU字(1枠有利・中枠不利・8枠やや浮上)。脚質を固定すると枠差はごく僅か(先行: 内20.1%/中18.7%/外19.1%)。内枠の微小上乗せのみでプロファイル条件化の価値は低い。脚質(逃げIV3.64)が支配的。
