高知2024+のテーマ4分析(騎手×調教師・馬場・血統)。レース情報に騎手・調教師・馬場・天候が全行で入っている(調教師108人・騎手128人)。

- **騎手×調教師コンビ(80騎乗以上)**: 打越勇厩舎が前残り中核(永森大/宮川実/吉原寛/井上瑛が前々率45-56%)。本命級かつ+EVは **永森大×打越勇(239騎乗・勝率37.2%・単回収144.9%)** と **林謙佑×田中守(117・23.9%・174.2%)**。宮川実×打越勇(498・36.1%)・赤岡修×田中守(724・34.1%)は高勝率だが回収は控除率付近。単回収順トップ(石本純×東原己312%等)は低勝率・小標本の人気薄ノイズで信頼不可。
- **馬場状態別の前残りバイアス(全データ・発走前判明で実戦可)**: 逃げ馬の単勝回収は 良199.6%/稍重196.2%/重205.9%/**不良215.4%**、勝率は良31.9%→不良**38.0%**。追込は不良で勝率2.4%・単回収23.9%とほぼ全滅。**不良/重ほど前残りが極端化**。kochi-scoreに馬場係数として組込み可能(不良で前を厚く・差し追込を嫌う)。
- **血統は使えない**: 父/母父データは高知2024+出走2,137頭中286頭(13%)のみ、かつJRA系転入馬に偏在。標本・交絡の両面で信頼不可、数値分析は見送り。馬情報テーブル充実後に再訪。

**Why:** 血統は期待されたが実データが薄く、馬場という別の高価値軸(前残りバイアスを増幅)を発見した。
**How to apply:** 馬場係数を [[kochi-backtest]] のスコアに追加検証する余地大。コンビは永森大×打越勇を本命根拠に。
