高知(2024-01〜2026-06, 28,824出走)の払戻金テーブルベース回収率検証(tools/kochi-roi.ps1)。オッズ未保存のため払戻金(¥100あたり)から算出。

- **バイアスの上限(今走脚質=事前購入不可)**: 今走逃げ 単勝回収204.7%/複勝130%(複勝率63%と安定)、今走先行 単勝92.7%。今走差し61.8%、追込41.9%。全頭買い基準は単勝73.7%/複勝69.0%(控除率の裏返し)。
- **実戦値(前走脚質=事前購入可)**: 前走逃げ+先行の単勝74.7%/複勝76.5%で、全頭買い基準とほぼ同じ。**市場は前走前々の馬を既に織り込んでおり、前走脚質の機械買いでは控除率を超えない**。
- **エッジの所在**: 「今走205% vs 前走75%」の差が、"今日逃げる/前に行く馬を群衆より正確に予測する"ことで取れる利益。妙味は1300〜1600mに集中(今走前々の単勝134〜140%)。
- 含意: [[chihou-keiba-db]] のkochi-scoreが狙うべきは前走脚質の有無ではなく、同型不在・騎手前付力・距離からの「今走前々」予測精度。
- **同型不在の効果(2024+)**: 前走逃げ馬は「同型(他の前走逃げ馬)不在=唯一」なら単回収77.5%/勝率20.6%、「複数」だと64.2%/15.5%(13pt差)。前走前々(逃げ+先行)馬は同型頭数で単調: 唯一85.2%・複回収83.6% → 2頭76.8% → 3頭以上72.8%。ただし最良でも控除率(≒100%)は未超。同型不在×騎手前付力×1300-1600mの重ね掛けが必要。

**Why:** バイアスの存在(知見)と「それで儲かるか(市場効率)」は別問題で、後者を金額で初めて検証した。
**How to apply:** 回収率の評価基準は単勝73.7%/複勝69.0%(全頭買い)。これを超えるかで戦略の良否を判断する。
