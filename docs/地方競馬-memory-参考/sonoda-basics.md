園田競馬の基礎分析(2022-2026、DB=地方競馬)。**Why:** 高知・大井で構築した軸有力度/馬券検証([[keiba-rating-system]])を園田へ適用する前提。**How to apply:** 検証は必ず複数年(各年4-6月窓)で。園田は年度間ブレ大。

## 規模・主力条件
- 充実は2022以降(各年≈1,600R/年、2016-21は疎で除外)。最新2026-06-17。
- 距離: **1400mが主力(4,770R)**、次1230m(872)/820m(794)/1700m(357)/1870m(271)。
- 条件中核: サラブレッド系 3歳以上 一般 / 4歳以上 一般。1400m一般で過半。

## 脚質バイアス(序盤位置early=最初の非ゼロコーナー、IV=脚質別勝率/全体平均)
強い前残り。距離が短いほど極端:
- 820m: 逃げIV5.42(勝率51.8%!)/先行1.33/差し0.48/追込0.06
- 1230m: 逃げIV2.85(27.7%)/先行1.39
- **1400m(主力): 逃げIV2.94(勝率29.7%/複勝57.9%)/先行1.45(14.7%)/差し0.90/追込0.35**
- 1700m: 逃げIV2.30、1870m: 逃げIV1.92(長いほど緩む)
- 四角(C4)位置が決定的(大井[[oi-kyaku-course]]と同型): 1400mで前→前28.6%・中→前(まくり)29.7%・後→前36.2%。四角で前に居れば序盤位置問わず勝つ。後方→四角前は稀(735/47k)。

## 枠順(馬番相対 内/中/外)
スプリントは**外枠やや有利**(前残りなのに外有利が園田の癖):
- 820m: 外12.5% vs 内7.1%(明確) / 1230m: 外11.5% vs 内8.1% / 1400m: 外11.3% vs 内8.9%
- 1700m+は差消失(母数小)。

## 主力騎手(1400m、rides>=150、2022-2026)
- **吉村智=支配的**: 2,719騎乗、前付け率25.1%・勝率27.1%・複勝60.0%(前にも行くし勝つ)
- 小牧太: 勝率26.1%(843、エリート) / 田中学18.9% / 下原理18.5% / 廣瀬航14.5%
- 長谷駿(前付27.2%/勝7.8%)・濱尚美(28.3%/7.8%)=前に行くだけで勝てない例。前付け率と勝率は別物。
