川崎([[kawasaki-kickoff]])の基礎分析(脚質/四角/枠/騎手)と軸有力度バックテスト結果。期間2024-01〜2026-06、tools(-Venue 川崎)。**Why:** 大井([[oi-kyaku-course]])の枠組みが川崎へ移植可能か検証。**How to apply:** 川崎は軸選定向き、回収は払戻近似で100%割れ前提。今走逃げ・前付け馬を軸に。

## 規模(確認済)
- N'川崎' 通年開催、2022以降充実(766-775R/年)、最新2026/06/17。三連単あり=馬券BT可。
- 距離: 1400m主力(1345R)>1500(777)>900(668)>1600(334)>2000(228)。

## 基礎: 脚質・四角・枠・騎手(oi-kyaku-course.ps1 -Venue 川崎)
- **極端な前残り**(大井より強い)。逃げIV: 900m=4.45 / 1400m=2.96 / 1500m=3.00 / 1600m=2.54。先行IV 1.6-1.8。差し0.7-0.9、追込0.2-0.45。短距離ほど極端。
- **四角位置が決定的**: 前→前 勝率25-33%。まくり(中後→四角前)も強力で**1400m以上で有効**(1400m 中→前26.8%/後→前33.8%、1600m 後→前30.8%)。900mは短く後方まくり不成立(後→前 n33のみ)。前→中後は▼大幅後退(ほぼ消える)。
- **枠は内やや有利だが小**(1400m 複勝 内.328/中.298/外.280、勝率 内.106/外.093)。脚質>>枠。
- **騎手前付け力&勝率TOP**(全距離200走+、勝率順): 御神訓27.2%(前付.418)、森泰斗24.1%(前付.448)、矢野貴21.3%、笹川翼18.9%、野畑凌18.0%(逃率.214最高)、山崎誠15.0%。高勝率騎手=高前付けで前残りと整合(=軸向き、単勝は織込済前提)。

## 軸有力度バックテスト(axis-backtest.ps1 -Venue 川崎、2年窓)
- 標準重み **h2h0.5/脚質0.2/騎手0.2/枠0.1** が頑健。直近(25-09〜26-06)勝率27.5%/複勝54.7%/単回収77、前年(24-09〜25-06)22.2%/52.9%/67.4。**2年平均≒勝率25%/複勝54%/単回収72%**。
- h2h単独(23.6%/20.4%)から脚質+騎手+枠で明確に向上=大井/高知と同じ較正が川崎でも妥当。
- **クラス補正は不採用**: 単回収を改善する行が年で入れ替わる=過学習ノイズ。[[keiba-rating-system]]の結論通り据え置き。
- 単回収<100%=南関の効率性([[oi-jockey-trainer]])と整合。価値は「堅い軸」。
