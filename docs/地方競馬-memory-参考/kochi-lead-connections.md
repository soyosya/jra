高知2024+で調教師・馬主・騎手の三者関係が「今走逃げ(early_pos=1)」をどう規定するか分析(全体逃げ率10.4%)。「上乗せ」=組合せ逃率−騎手単体逃率。

- **調教師の戦術効果**: 打越勇厩舎は宮川実/永森大/妹尾浩/井上瑛/吉原寛の5騎手全員で+6〜13ptの逃げ上乗せ(騎手でなく厩舎の方針)。東原己×石本純 +16.9pt(石本単体7.5%→24.5%, 逃時単回収480%)、國澤輝×阿部基 +11.0pt(逃時438%)。
- **馬主の独立効果(ご質問の核心)**: 「前へ行く馬主」が実在。中村智幸は宮川実/井上瑛/吉原寛の3騎手全員で逃げ率激増(+15〜23pt、宮川実で46.2%)。ウエスト.フォレスト.ステイブルも郷間勇/山崎雅/岡村卓/岡遼太の4騎手で一貫上乗せ。
- **三者(調教師×馬主×騎手)が最強シグナル**: 國澤輝×JPN技研×阿部基 39.0%(+28.5)、東原己×小橋亮太×石本純 37.2%(+29.7)、田中守×須田靖之×林謙佑 36.6%(+26.6)、打越勇×中村智幸×宮川実 46.2%。低逃率騎手が前傾厩舎+馬主で逃げ率37-39%に激変。
- **市場非効率**: 低逃率騎手×前傾厩舎/馬主は逃げた時の単回収400-480%。穴の逃げ激走の温床(市場が織り込まない)。
- 注意: 馬主と厩舎は相関(中村智幸の馬は主に打越勇)。三者標本は40-90走と小さい。ただし同一馬主が複数騎手で一貫高逃率という横断整合性が効果の実在を裏づけ。

**深掘り(前走重ね・LOO逃げ率)**: 「強コネ」=三者延べ>=30走&LOO逃げ率>=20%の馬(1,176走)は逃げ率30.8%(非強コネ9.3%の3.3倍)。強コネ×前走逃げ=55.1%、×前走先行=34.2%、×前走前々×同騎手=46.9%が最強の逃げ予測。後ろ脚質の強コネ馬が逃げると激アツ(差し起点逃時単回収392%/追込400%)=穴の逃げ温床。前走上り3Fが遅い方が逃げ率・配当とも高い(遅い34%/316% vs 速い31%/131%)。枠順・騎手継続は強コネ内では無意味。

**Why:** [[kochi-lead-transition]]で「乗替先が前付け騎手なら逃げ倍増」と判明、その延長で馬主・厩舎にも独立した前傾シグナルがあることを確認。
**How to apply:** 前傾コネクション(厩舎:打越勇/東原己/國澤輝、馬主:中村智幸/ウエスト.フォレスト)を逃げ予測の加点に。無名騎手騎乗時ほど妙味大。[[kochi-backtest]]への組込み候補。
