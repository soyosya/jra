園田 追加調査(2022-2026、記述的回収分析)。基礎[[sonoda-basics]]・軸/馬券[[sonoda-axis-bet]]の補完。**How to apply:** 軸有力度に下記を加点/減点材料として併用。ツール: tools/sonoda-extra.ps1。

## 馬場差(1400m 逃げ勝率)
良29.0 / 稍重27.4 / **重36.3** / 不良32.8。**道悪(特に重)で前残り強化**(良→重で+7pt)、差し/追込は道悪で勝率低下。→重・不良の日は逃げ軸を一段重く([[kochi-jockey-trainer-track]]の不良前残りと同方向)。

## 調教師(年別で頑健性確認済)
- **永島太=頑健な実力厩舎**: 勝率5年とも≈20%(基準10%の倍)、前付け17%。**単勝回収108%は2023年(228%)の一発依存**で他年51-95%=単勝プラスではない→用途は軸・複勝の信頼補強。
- **吉見真=新興妙味候補**: 直近2年(2025:142%/2026:130%)で単回収100%超&勝率上昇、ただし2年のみ=継続観察。
- 雑賀伸(2022の178%依存)・住吉朝(2022のみで以降消失)の高ROIは単年依存=不採用([[oi-roi-backtest]]の過学習回避と同じ規律)。

## 馬主
ROI上位は小標本(n84-194)ノイズ中心で頑健性低く採用見送り。兼子尚也(n118・勝率35.6%・前付け34%・単160%)のみ逃げ×高勝率で突出だが標本中。

## 斤量・減量騎手 = 妙味なし(むしろ軽視材料)
減量記号あり: 勝率6-7%・単回収59-68%/複60-64%。無印: 勝率10.6%・単64-73%/複67-69%。**減量騎手は勝率も回収も劣る**=見習いの斤量恩恵は園田で技量差を補えない。

## 距離替わり・昇降級(基準 単67/複68%)
- **距離延長は全般に不利**(勝率6-10%・回収低)=園田は延長馬を軽視/消し。
- **距離短縮は買い**: 短縮×同級=単回収79-95%(4年)、**短縮×降級=複回収74-84%で5年安定(最も堅い妙味)**。
- 降級馬は勝率明確に高い(同距離降級12-13% vs 同級8.7%)。同距離昇級も勝率高いが回収は織込済(軸向き)。
