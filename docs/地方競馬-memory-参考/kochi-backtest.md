高知の改良スコア tools/kochi-backtest.ps1。特徴量=前走脚質・同型不在(レース内の前走前々頭数)・騎手前付力(学習期間のみで算出しリーク回避)・馬場差補正後持ち時計([[kochi-track-variant]]のδ)・距離。特徴量はtools/_kochi_feat_cache.csvにキャッシュ(-Refreshで再抽出)。

学習 2024-01〜2025-09 / 検証 2025-10〜2026-06(884レース)、各レースで最高スコア馬を単勝・複勝¥100購入。

- **本命1点 単勝回収**: 現行相当(脚質+騎手+持時計)91% → 改良(+同型不在+距離)101% → 改良+価値コネ+馬場 **108.6%**(複勝84.0%)。全頭買い基準73.7%を大きく上回る。
- **コネクションの正しい使い方(重要)**: 三者(調教師×馬主×騎手)逃げ率を素朴に加点すると的中率は上がる(27.6%)が回収率は91%に下落=強コネは人気の前付けで市場が織込済([[kochi-lead-connections.md]])。「隠れた前残り」= 前走差し/追込なのに強コネ(市場が逃げを予想しない穴)にのみ加点すると回収108.6%。馬場は不良で追込を減点。価値は"市場が見落とす前残り"に宿る([[kochi-roi-payout]]と同型の教訓)。
- **選別力**: 本命の単勝的中26〜34%(基準10.2%の約3倍)、複勝的中50%。
- **閾値スイープ(価値コネ版)**: スコア≥3で675レース・単回収119.4%、≥4で378レース・121.9%、≥5で158レース・131.0%。閾値で絞る運用が良い。
- **頑健性**: 価値コネ版の四半期別は25Q4=118.4%/26Q1=90.5%/26Q2=122.8%(26Q1も従来74.7%→90.5%に改善)。単勝回収95%CI[83.3%,139.8%]で下限上昇も100%はまたぐ(単勝の高分散ゆえ)。複勝は常に控除率割れで妙味なし。
- **結論**: 強力な選別器だが「統計的に確実な黒字」とは言えない。点推定は+EV。実戦トラッキングで継続検証する段階。

**Why:** バイアス・同型不在の知見([[kochi-roi-payout]])を実際の予測スコアに統合し、リーク無し検証で初めて回収率100%前後を実証。ただし単勝は高分散。
**How to apply:** 本命選定はスコア≥3で絞る。回収率100%超は確証でなく点推定として扱い、過信しない。
