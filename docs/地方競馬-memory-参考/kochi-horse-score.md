高知の馬軸 勝率スコア tools/kochi-horse-score.ps1(キャッシュ _kochi_horse_cache.csv、-Refresh で再抽出、-ShowRace yyyy-MM-dd で各馬の勝率%表示)。学習2024-01〜2025-09 / 検証2025-10〜2026-06(900レース)。

特徴量(全て前走/前々走のファクト・リーク無し):
- 能力: 馬場差δ補正後の同距離自己ベスト vs 距離×クラス基準、前走 一着馬着差
- 体調: 前々走→前走の 上り3F改善 / 着差改善 / 四角通過位置の前進 / 前走馬体重増減の安定性(※増減の符号未確認のため|Δ|>=12を不安定として軽い減点に留める)
- クラス変動: 今回 一着賞金 vs 前走(降格+1.5 / 格上挑戦−1.0、前走圧勝なら緩和)
- コース相性: 同距離自己ベストの有無、今回馬場での過去複勝率
- 展開(Bのみ): 前走脚質 / 同型不在 / 騎手前付力 / コネクション隠れ前残り

**結果(本命1点・最高スコア)**:
- A:馬ファクトのみ … 単勝回収73.6%(全頭買い基準73.7%と同等)/的中16.7%。**能力・体調・クラス・相性は市場が正確に織込済で単独では+EVにならない**(高知市場は馬の強さに効率的、[[kochi-roi-payout]]と一貫)。
- B:展開統合 … 単勝回収**116.1%**/的中25.1%、複勝87.1%。馬ファクトを[[kochi-backtest]]の展開要素に足して108.6%→116.1%に向上(降格・補正タイム・体調トレンドが本命選定を精緻化)。
- **較正が極めて良好**: 予測勝率%≒実勝率(A: 12.4%→12.1%, 17.1%→17.3% 等。B: 11.5%→12.0%, 28.6%→29.4% 等)。出力する「勝率%」は信頼できる。
- 出力は「生スコア」+「レース内正規化した較正勝率%」の両方、モデルA/B両方。
- 注意: 賭けの本命選定は生スコア最大で行う(回収率はこれで算出)。較正勝率%は解釈用で、超高スコア帯は標本薄で僅かに非単調。

**Why:** 「馬を軸に勝つ可能性を数値で」の要望に対し、ファクト由来の較正勝率%を実装。馬の強さは当てられる(較正良好)が儲けの源泉は展開の歪みにあることを改めて定量化。
**How to apply:** 予想は B の勝率%で序列、妙味判断は「能力評価が高い(A高)のに展開で人気が割れる」馬を狙う。馬体重増減の符号はDBで要確認(現状|Δ|のみ使用)。
