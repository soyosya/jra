大井・1200m・一般別定・1着賞金100万 の勝ち馬プロファイル(本線4条件スコア0〜4)を、払戻金テーブル(単勝/複勝, 金額=100円あたり払戻)で2024-01〜2026-06にバックテストした結果(2026-06-15実施, 対象230レース/延べ3120頭)。スクリプトは tools/oi-roi-backtest.ps1。オッズ/人気はDB未保存のため価格フィルタは不可能。

**基準(全頭買い)= 単83.5% / 複68.3%**(これが控除後の期待値ライン。回収率はこれと比較して評価する)。

スコア別 単回収: score4=89.4%(勝率22.9%・複勝率50.8%)、score3=56.2%、score2=106.4%、score1=40.4%。
- **score4 はプロファイルとして本物**: 勝率22.9%(全体ベース約7%)で基準83.5%を+5.9pt上回る。ただし<100%なので単勝フラット買いでは非黒字。「勝率を当てる」力はあるが市場に織込まれ妙味は薄い。
- **score3(部分合致)は56%の地雷ゾーン**。中途半端に条件を満たす馬は買ってはいけない。
- **score2の106%は完全な外れ値**: 単勝10倍未満に限ると勝率0%・単回収10.1%。万馬券一発の幻。

**サブ条件(格下げ/休明け)は過学習。単独では信用不可**:
- 休明け(前走から43日以上)単回収233.9%は2025年(387.5%)の一発依存。2024年は64.3%で負け。
- 格下げ(前走が上クラス)単回収145.5%も、休明けとの重複21頭(≈291%)が正体。「格下げ∩休明け除く」にすると71%(基準割れ)で、年次も103/55/0%とバラバラ。
- サブ条件部分集合は n=40〜60 と小さく多重検定の罠。単年の高回収率は高配当一発か特定年への過学習を疑うこと。

**馬券効率も控除率超えず**: score4を軸にしたワイド/馬連も基準割れ。ワイド総流し63.7%(死に目を買い希薄化で逆効果)、相手をscore≥3に絞ると75.5%、軸相手とも本線score4に絞って84.0%まで戻るが単勝score4(92.7%)に及ばない。score4の高複勝率(50%)は妙味に変換できない(的中する相手も強く低オッズ)=単勝が最も効率的。

結論: このプロファイルは「勝率の高い馬の同定」には有効だが、払戻ベースで控除率を超える安定エッジ(robust >100%)は現データ・現条件では見つからない。実用上は score3 を避け score4 に絞るのが筋。+EVは馬の条件側でなく厩舎側にあった([[oi-jockey-trainer]] 荒山勝)。関連: [[kochi-roi-payout]](前残りバイアスも市場に織込済だった件と同型の結論)、[[chihou-keiba-db]]。
