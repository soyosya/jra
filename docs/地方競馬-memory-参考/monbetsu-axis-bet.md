門別の前残り軸スコアと検証。基礎は[[monbetsu-basics]]、枠組みは[[keiba-rating-system]]、比較[[kawasaki-basics-axis]]/[[sonoda-axis-bet]]/[[oi-roi-backtest]]。

## ツール(C:\temp\tools\、接続は[[chihou-keiba-db]]のkeiba_q.ps1経由)
- `monbetsu-features.sql` → mb_features.csv: 競走結果ベースの過去走特徴量(prevstyle/門別逃げ・勝・複回数/gap/prevdist/単複払戻)。**着順>0必須なので未来レースは入らない**。
- `monbetsu-backtest.ps1`: 学習年(既定2022-23)で騎手・調教師の勝率を作り、テスト年(既定2024-25)で各レース最高スコア=本命を選び勝率/複勝率/単複回収を年別表示。
- `monbetsu-card.sql` → mb_card.csv: 指定日の**出馬表(レース情報)**に各馬の過去走特徴を結合(未来レース予想用、開催日<当日でリーク回避)。
- `monbetsu-today.ps1 -Date 'YYYY/MM/DD'`: mb_card.csvをスコア化しR別に◎○▲とフラグ(前付騎手/前傾厩/前傾主/長休明/中1週△/延長△)表示。
- スコア式: 2.0*前走脚質(逃1.0/先0.55/差0.15/追0/初0.30)+1.6*馬門別逃げ率+1.2*馬門別勝率+5.0*騎手勝率+3.0*調教師勝率+0.5*外枠+gap補正+延長補正。重みは経験的。

- **本命 勝率28.5%・複勝58.4%・単回収83.1%・複回収82.1%**。年別: 2024=28.7%/58.9%/75.2、2025=28.3%/57.9%/90.8。勝率の年ブレ小=過学習なし。
- 回収は単複とも100%割れ([[oi-roi-backtest]]/[[kawasaki-basics-axis]]/[[sonoda-axis-bet]]と同様、軸は的中用で利益保証ではない)。勝率28%は park中で良好。
- ライブ確認 2026-06-17 R1: ◎ルアシュエル1着・▲キモサベ3着、R2: ◎カスタネアローズ2着(複勝)=BTと整合。
- **2026ホールドアウト(2022-25学習→2026の179R)**: 勝率32.4%/複勝67.6%/単回収77.3%/**複回収95.0%**。最新レジームでも頑健(むしろ的中力は過去最高水準)=ロジック修正不要。**複回収95%は単勝より複勝/ワイド軸が妙味**を示唆([[sonoda-axis-bet]]と同傾向)。2026は春のみ部分標本で年度ブレ留意。

## 能力統合(走破時計・上り3F)2026-06-17追加 ← 現行版
- **レース内z-score**で能力を測定: 各過去レースで `z=(レース平均時計−自馬時計)/レース内SD`(z>0=平均より速い)。**開催場所・距離・馬場・開催日を自動相殺**(時計の絶対正規化不要)。同様に上り3Fもz化。リーク回避(出走前までの平均、現走は除外。features/cardのSQL CTE `z`/`abil`)。field≥5 & SD>0 のみ。
- スコアに `+WTime*ab_time + WUp3*ab_up3` を追加(既定 **WTime=2.0 / WUp3=1.0**)。features/card SQLに ab_time/ab_up3/ab_best/ab_n を追加済。backtest/today/nagashi/score-race.ps1 が対応。
- **効果(ホールドアウト)**: 本命勝率 28.5%→**34.0%**・複勝 58.4%→**66.2%**(2024:34.5/2025:33.4)、2026は32.4%→**39.7%**。全年で向上=頑健。**ただし単回収は83→74に低下**(時計上位馬は人気・低配当で市場に織込済)。複回収は82〜88で横ばい。→**当てる用途は明確改善、儲け用途は中立〜不利**。
- 副次効果: 前回指摘の弱点(履歴薄の馬がデフォルト逃げ率0.30で過大評価)を時計zが矯正。例 2026-06-17 R8で1番サクセスビービー(時計z+0.91=最速)が7位→2位に浮上、逆に門別1走の10番(z−0.84)が5位→9位へ妥当に降格。
- `monbetsu-score-race.ps1 -Race N`: 1レースを能力なし/ありで並べ替え比較(検証・説明用)。
