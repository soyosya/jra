## 脚質(1000/1200m, 四コーナー位置で判定。1000/1200mは三・四角のみ取得)
- **前残り極端**: 逃げ(4角先頭)勝率38.4%/複勝70.6%(**逃げIV≒3.8**)、先行14.7%、差し5.7%、追込0.7%。大井1200m(IV3.64)を上回る。四角位置が勝敗を決める。
- 枠: **外枠有利**(外勝率11.5% > 中9.9% > 内8.5%)。スプリント外有利は[[sonoda-basics]]と同傾向。
- 馬場: 悪化で前残り強化(逃げ勝率 不良44.7% > 重40.0% > 良37.7% ≈ 稍重37.4%)。[[kochi-roi-payout]]/[[sonoda-extra]]と同方向だが効果は中程度。

## 騎手(1000/1200m, 騎乗≥300)
- **落合玄太が支配的**: 勝率19.4%・複勝45.5%・**逃げ率26.9%(前付け力で突出)**。前残り場×前付け騎手の典型軸。
- 次点 石川倭(勝率19.4%・逃げ率18.2%)。以下 桑村真15.2%・服部茂14.8%。勝率は逃げ率/前付率と強く相関。

## 逃げ予測・コネクション・妙味(2022-2025)
- **前走脚質が最大予測子**: 前走逃げ→今走逃げ率38%・勝率21%、前走追込→逃げ率3%・勝率3%。[[kochi-lead-transition]]と同じ。
- **前傾厩舎**: 田中淳(逃げ率25%・勝率23%)・角川秀(22%/17%)・安田武・川島洋・小野望。**前傾馬主**: 菅野守雄(逃げ24%・勝率21%)・橋本富夫・土橋正雄・平井克彦。逆に北原大史/門別敏朗系は差し追込。
- **妙味**: 長期休養明け(3ヶ月超)は単回収85=過小評価で最良。中1週(≤10日)は勝率4.6%で危険。**距離延長は弱い**(勝率7%・単回収58)、短縮の妙味は門別では薄い(同距離と差なし=園田と違う)。
