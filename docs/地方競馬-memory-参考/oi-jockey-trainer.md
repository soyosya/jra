大井(2024-01〜2026-06)の騎手・調教師別 単勝回収率(払戻ベース)。スクリプト tools/oi-jockey-trainer.ps1(-Detail <名前> で年次分解)。リアルタイムオッズは大井0件・血統紐付け14%のため、オッズ/血統軸は大井では不可。騎手/調教師はレース情報に全件あり。

**調教師 荒山勝**: 全体650件・勝率28.5%・単回収126%、年次131.8/132.4/98.4と桁違いに安定。安定の理由は高勝率(多数の中配当勝ちでROIが決まる)で、他厩舎の高ROI(少数の万馬券依存)とは安定の質が違う。**ただし深掘りすると単勝の+EVは見かけほどでない**:
- 距離別: 1200=91.7%/1400=100.1%/1600=118.3%は堅実だが、全体126%は1800m(264.9%, n53, 最高96倍)の外れ値で嵩上げ。
- 事前に使える前走脚質別: 前走逃げ(勝率38.5%)単回収89.1%/前走先行(30.2%)96.1%=織込済。高ROIの前走差し146%・追込213%は77倍/96倍の一発依存。
- 「今走逃げると勝率54.7%・単回収178.6%」は四角先頭という結果論で事前には使えない。
→ 荒山は「複勝率60%で買えば来る・軸向き」の高勝率厩舎だが、単勝で儲かるほどの過小評価ではない(クリーンな先行馬は95%前後)。前回の「唯一の頑健な+EV」評価は撤回。

**他の南関主力場(船橋/川崎/浦和)にも頑健な過小評価厩舎なし**: 高ROI厩舎は低勝率+超高配当一発(川崎八木喜140%=261倍依存/浦和横山保206%=250倍依存)。中勝率の候補(船橋渡邊貴114%・浦和小林真115%・船橋小久智)も年次は33→142→224%等と乱高下=その年の万馬券次第。高勝率厩舎は軒並み95-100%で織込済。南関の単勝市場は4場とも効率的。

**過学習・分散の罠(いずれも信用不可)**:
- 低騎乗・低勝率・超高回収(福原杏272騎乗・勝率2.6%・241%、高橋昭170%等)は高配当一発の分散。
- 騎手 吉井章 単回収112%@1388騎乗は一見大サンプルだが、2024(166.9%・最高263倍)一発依存で2025/26は78/67%と負け越し=過学習。
- トップ騎手 御神訓(勝率20.2%)は単回収86%で人気先行=過大評価。強い騎手ほど織込済。

教訓: 払戻ベースのリーダーボードは必ず年次分解+最高配当で外れ値を除外して判断する([[oi-roi-backtest]]のサブ条件過学習と同型)。高知の[[kochi-jockey-trainer-track]]と同方向(強い人気どころは織込済、妙味は別)。関連 [[oi-kyaku-course]]。
