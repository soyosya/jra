高知の日次馬場差補正 tools/kochi-track-variant.ps1(2024+)。

- **方法**: 期待タイム=「距離×一着賞金(クラス)」の平均勝ち時計(標本<8は距離中央値で代替)。破損タイムは妥当速度域 dist/18〜dist/11 [m/s] + 距離中央値±12% で除外。日次馬場差 δ = その日の各レース偏差(勝ち時計−期待)の中央値。補正後タイム = 生タイム − δ(正のδ=時計が掛かる重い日)。
- **結果(2,934勝ちレース・273開催日)**: δは最速−4.18s〜最重+3.60s(レンジ7.78s)。同距離でも日で7秒近く時計水準が違う。
- **検証**: 同一(距離×クラス)セルの勝ち時計の加重平均SDが補正前1.620s→補正後0.905s(44.2%縮小)。日次δが日間バラつきを的確に捕捉。
- 2024+の勝ち時計に破損データは無し(0除外)。破損は非勝ち馬や2024年以前に存在。
- 用途: [[chihou-keiba-db]] のkochi-scoreの持ち時計評価を「補正後自己ベスト」に置換する土台。[[kochi-roi-payout]] の予測精度向上に寄与。
- 注意: δが勝ち時計中央値で作られ検証も勝ち時計を使う点でやや自己参照的だが、中央値ゆえ単一レース影響は小。
