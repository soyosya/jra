川崎([[kawasaki-kickoff]]/[[kawasaki-basics-axis]])の馬券BT(ステップ4)と妙味検証(ステップ5)。2024-01〜2026-06、年別。単勝回収のbaseline≈0.75-0.80(控除後)。**Why:** 南関効率性([[oi-jockey-trainer]])が川崎の馬券・妙味にも当てはまるか。**How to apply:** 回収プラスは出ない前提、価値は「堅い軸」。馬体重フラグは川崎では信頼しない。

## 馬券BT(trio-backtest.ps1 -Venue 川崎)
- 3連複 軸1相手5(10点): 直近(25-09〜26-06)的中19.4%/回収49.4%、前年(24-09〜25-06)22.4%/41.1%。**2年平均 的中約21%/回収約45%**。広く買い過ぎで控除前(理論75%)を大きく割れ。点数を絞るのが筋だが南関効率的で大幅改善は見込み薄。複勝/ワイド軸が現実的(園田[[sonoda-axis-bet]]も複勝84%が最良)。

## 妙味検証(年別単勝回収)
- **馬体重減は妙味なし(キックオフ想定外の重要発見)**: 1走版 馬体重増減≤-10kg は3年で回収0.58/0.53/0.94・勝率フラット(~0.09)。2走版(今走−前々走≤-10)も回収0.45/0.74/1.10・**勝率0.084/0.089/0.087=全体平均と差なし**。高知のような勝率2倍効果は川崎では出ない。年ブレ大。→**川崎では-10kgフラグを軸根拠にしない**。[[keiba-rating-system]]の「川崎も弱い正」は本場別多年検証では実質中立=訂正。today-picksで点灯しても川崎では割引く。
- **出走間隔(堅さ要素)**: 中5-6週(勝率0.089-0.109)・7-13週(0.123-0.142)が**勝率安定して高い**=堅い軸補強。**≤2週(詰め)は勝率低い**(0.05-0.07)→[[keiba-rating-system]]の前6着×続戦減点と整合。休養明け>13週も悪くない(勝率0.09-0.12)。回収はいずれも基準内(妙味ではなく堅さ)。
- **距離変化**: **同距離が最も堅い**(勝率0.092-0.101で安定、回収~0.74)。**大幅短縮≥200mは不振**(勝率0.05-0.07/回収0.33-0.60)→軽視。延長は年ブレ大で妙味とは言えない。**園田の「短縮が堅い妙味」([[sonoda-extra]])は川崎では成立せず(むしろ逆)**=場ごとに違う。
- **馬場×前残り**: 1400m逃げ勝率 良0.272<稍重0.286<**重0.320**=重馬場で前残り強化(前付勝率も重0.222最高)。園田・門別と同傾向。不良は逃頭n38で標本小・保留。

## 総括
川崎は[[kawasaki-basics-axis]]の通り軸選定向き(2年平均勝率25%/複勝54%)だが、馬券回収・妙味は南関効率性で**プラス材料に乏しい**。実用フラグ候補は「中5-13週の堅さ」「同距離の堅さ・大幅短縮の軽視」「重馬場の前残り強調」。馬体重-10kgは川崎では非推奨。
